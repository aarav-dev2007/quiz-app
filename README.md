# Quiz App

A console-based General Knowledge Quiz Application built using Python.  
It includes both **Admin** and **User** modes with full quiz management features.

# Admin Panel
- Add new questions
- Delete existing questions
- Update options and correct answers
- Search questions
- View total number of questions

# User Mode
- Randomized questions each time
- Input validation (A/B/C/D only)
- Score calculation
- Percentage display
- Performance feedback

# Tech Used
- Python (Core)
- Random module

# How to Run
- Go to the Releases section of this repository
- Download the .exe file
- Run the file on your system

# Platform Compatibility

The provided `.exe` file has been built specifically for **Windows operating systems** and may not run on macOS or Linux.

---

## Running the Project on Other Operating Systems

If you are using macOS or Linux, you can still run the project using Python:

### Steps:

1. Make sure Python is installed on your system
2. Clone this repository or download the source code
3. Open a terminal in the project folder
4. Run the following command:

```bash
python quiz.py
```

---

## Notes

* Windows users can directly run the `.exe` file from the Releases section
* Non-Windows users should use the Python method described above
* Ensure all required files are in the same directory before running the script
